# E-commerce Platform with Stripe Integration

[![Laravel](https://img.shields.io/badge/Laravel-FF2D20?style=for-the-badge&logo=laravel&logoColor=white)](https://laravel.com)
[![Stripe](https://img.shields.io/badge/Stripe-635BFF?style=for-the-badge&logo=stripe&logoColor=white)](https://stripe.com)
[![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)](https://tailwindcss.com)

A professional e-commerce solution developed for client needs, built using the Laravel framework. This system focuses on secure transaction handling, inventory management, and a robust checkout flow.

---

## Key Features

* **Advanced Stripe Integration:** Secure payment processing with Stripe Checkout Session, supporting IDR currency.
* **Dual-Path Checkout Logic:** Flexible system allowing users to purchase directly from a product page or through a consolidated shopping cart.
* **Complex Order Management:** Backend logic for tracking order status from 'pending' to 'success'.
* **Customer Address System:** Support for multiple shipping addresses per user account.
* **Administrative Dashboard:** Full-featured panel for managing products, monitoring sales, and handling order logistics.
* **Custom Frontend Architecture:** Built with Tailwind CSS and advanced animation libraries for a high-end user experience.

---

## Tech Stack

* **Core Framework:** Laravel 10.x / 11.x
* **Database:** MySQL (Modular schema design)
* **Frontend:** Blade, Tailwind CSS
* **Payment Gateway:** Stripe API
* **Animations:** GSAP, ScrollTrigger, & Lenis Scroll
